## Prerequisites

- node >=16.6.0

## Install

```sh
npm install
```

## Usage

#### Edit you `.env` file:

```sh
cp .env.example .env
````

#### Start postgres

```shell
docker-compose up -d postgres
```

#### Now you can start the bot

```sh
npm run start
```

### Run in Docker

```shell
docker-compose up -d
```
